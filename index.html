Interactive UX/UI Development Guide

"For most users the UI is the system."

Jesse James Garrett's Five Planes of User Experience Design provides a comprehensive framework for creating user-centered digital products. His model breaks down the UX design process into five interconnected layers, moving from abstract to concrete. This Interactive UX/UI Development Guide shows the model from two perspectives: The "Product Arc" and the "User Arc".
These two arcs run in conjunction with each other, over-lapping throughout the design process. The Product Arc begins with a problem, or product idea, and the User Arc begins with the identification of a possible end-user.
In the User Arc we move through the planes in reverse, because this is how the user experiences your product.
This is a tool for developers, designers, and product managers to understand the UX design process and create better user experiences. Click on the elements to explore each plane in detail, and shift perspective to move between arcs.
